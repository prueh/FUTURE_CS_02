# Phishing Awareness & Social Engineering Defense (Task 02)

## Project Overview
This project was developed for the *Future Interns Cybersecurity Internship* to address the #1 vector for data breaches: Phishing. I designed a professional, high-impact awareness guide to train employees on identifying and reporting sophisticated social engineering attempts.

## Awareness Modules
* *The Anatomy of a Phish:* Breaking down the sender address, artificial urgency, and malicious attachments.
* *Link Validation:* Teaching the "Hover Technique" to inspect destination URLs before interacting.
* *Credential Protection:* Highlighting why legitimate organizations will never ask for passwords via email.
* *Incident Response:* A simplified 3-step protocol (Disconnect, Do Not Enter Data, Report) to minimize damage after a suspected click.

## Technical Implementation
* *Clean Professional Theme:* Developed with a White & Blue "Executive" color palette for corporate environments.
* *Frontend Stack:* Built using HTML5 and CSS3 for a seamless, browser-based presentation experience.
* *Interactive Elements:* Utilized Font Awesome icons and CSS transitions to emphasize critical "Red Flags."
